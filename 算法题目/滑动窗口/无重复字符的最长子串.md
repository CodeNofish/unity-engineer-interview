给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**滑动窗口**
```C#
public class Solution {
    public int LengthOfLongestSubstring(string s) {
        int n = s.Length, res = 0;
        var hash = new HashSet<char>();
        int head = -1;
        char[] chars = s.ToCharArray();
        for (int tail = 0; tail < n; ++tail) {
            char tailChar = chars[tail];
            // 每一步滑动，确保tail必须加入窗口
            // 从head开始清空 导致tail重复的
            while (hash.Contains(tailChar)) {
                head++;
                char headChar = chars[head];
                hash.Remove(headChar);
            }
            // tail元素加入窗口，统计此时的长度
            if (!hash.Contains(tailChar)) {
                hash.Add(tailChar);
                res = Math.Max(res, tail - head);
            }
        }
        return res;
    }
}
```


**暴力解**
```C#
public class Solution {
    public int LengthOfLongestSubstring(string s) {
        int n = s.Length, res = 0;
        var hash = new HashSet<char>();
        for (int i = 0; i < n; ++i) {
            for (int j = i; j < n; ++j) {
                char ch = s[j];
                if (hash.Contains(ch)) {
                    res = Math.Max(res, hash.Count);
                    hash.Clear();
                    break;
                } else {
                    hash.Add(ch);
                }
            }
        }
        res = Math.Max(res, hash.Count);
        return res;
    }
}
```
